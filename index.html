    Google Drive Storage

    Data tersimpan otomatis di Google Drive Anda — aman, bisa diakses dari HP manapun

    —
    ⚙️ Konfigurasi
    1
    Buat Google OAuth Client ID

    Buka Google Cloud Console → Buat project → Aktifkan Google Drive API → Buat OAuth 2.0 Client ID (Web Application) → Tambahkan URL file ini ke Authorized JavaScript origins.

    2
    Login ke Google

    Setelah Client ID tersimpan, klik tombol Login di bawah. Izinkan akses ke Google Drive.

    3
    Simpan & Muat Data

    Data disimpan sebagai file cost_listrik_data.json di root Google Drive Anda. Bisa dibuka dari HP iPhone, Android, atau komputer manapun.

    📊 Status Data Lokal
    Belum ada data diinput.

    Cara Kerja Penyimpanan

    • Data disimpan sebagai file JSON di Google Drive Anda
    • Nama file: cost_listrik_data.json
    • Buka aplikasi dari HP → Login → klik Muat dari Drive
    • Data bulan lama tersimpan aman, tidak tertimpa
    • Setelah input data baru, klik Simpan ke Drive

    Tips Akses dari HP

    iPhone Safari: Buka file → Share → Add to Home Screen
    Android Chrome: Buka file → ⋮ → Add to Home Screen
    Jadi shortcut di layar utama seperti aplikasi.
    Penting: File harus dihost di web server / GitHub Pages / Google Sites agar OAuth berfungsi di HP.
